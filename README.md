# Cabbagelang
This is the official repository of Cabbagelang Programming Language.

## Installation

### Compile

> Windows

`make windows`

> Unix-like system (Linux, MacOS, etc)

`make unix`

### Install

> Windows

`install.bat`

> Unix-like system (Linux, MacOS, etc)

`chmod +x ./install.sh`

`./install.sh`

## Dependences

- **Raylib**

- **LibCURL** *Linux*

- **LibEDIT** *Linux*
